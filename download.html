How to connect to L2SamuX

Step 1: Install Lineage 2 High Five (Already got High Five? Move on to Step 2)

Using the link above, or click here, you can start to download the Lineage 2 High Five game files.
To install these, simply run the installer, click Next -> Next -> Next -> Finish, and you can move on to Step 2.

Step 2: Install L2SamuX system patch files

Download our system patch files using the button above.
Place it in your Lineage 2 folder and "extract here".


Make sure to allow it to overwrite all.

You should now be able to run l2.exe which is located in lineage 2/system folder.

When run game you might get this message

Simply click on OK and the game should successfully launch.

We hope you enjoy your stay!
